## 5. 预处理决策依据

1. **统一尺寸**：原始尺寸差异大，统一 resize 至 224×224 以适应预训练模型输入。
2. **数据增强**：训练集采用 RandomResizedCrop、RandomHorizontalFlip、ColorJitter 防止过拟合。
3. **无需清洗**：无损坏文件，格式统一，可直接进入 DataLoader。
4. **无需重采样**：类别 1:1 均衡，交叉熵损失无需加权。
